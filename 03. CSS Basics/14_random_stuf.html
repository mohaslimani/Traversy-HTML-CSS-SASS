    • !Important // selectors order

    • in css, u can list multiple options for one property, and separate them with comma (,)
    • font-family: Georgia, serif; ::::: "Georgia" will be used, but if it not exist in the browser "serif" will be used.
    • class selectors will override element selectors.
    • CSS parse the .class last, so the order between .class selectors and element selectors wont make difference.
    • if u put !Important after a property that property will override every other one, no matter the order or the selector used.
    • this property helps using other css Frameworks (like bootstrap).
