flex & flex-items properties.

direction : row(default main axis)

the flex items can take 3 possible properties
and will affect the flex only horizontally

  • flex-shrink : (0 noshrink / 1 ..) : Sets the flex shrink factor.
  • flex-grow : (0 noGrow / 1 ..) Sets the flex grow factor.
  • flex-basis : number / a factor for each item (what fraction it can take per item).

flex-shrink

if u put this shit to 0, its original width will be so fixed.
else it will shrink by a factor.

  • shrink 0 : will always hold its width, even if it pushes the other boxes outside of our page (horizontal scroll).
  • shrink 1 : will take its 300px until there is no more spase then it will start shrink.
  • shrink 3 : like shrink 1, but will shrink more agressivly.
