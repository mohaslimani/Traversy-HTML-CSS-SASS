html Element

html Element is the root element

html element has default computed/fixed sizes for font / margin / padding.....

u can change those values using an html selector. html { }

    Layout
  • margin-bottom: 0px;
  • margin-left: 0px;
  • margin-right: 0px;
  • margin-top: 0px;
    Text
  • font-size: 16px
    Other
  • padding-block/inline-start/end : 0px

Browser Setting

certain browser setting such as

  • font size option in chrome

    • will change the html default values.
