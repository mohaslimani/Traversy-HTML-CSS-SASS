There are two display values: block and inline

  • A block-level element always starts on a new line and takes up the full width available
    • div is a block element
    blocks always starts on a new line.
  • A block level element has a top and a bottom margin, whereas an inline element does not.
  • An inline element does not start on a new line and it only takes up as much width as necessary
    • "a" is an inline element "a" is an inline element
    Note: An inline element cannot contain a block-level element!
  • also we have something called inline-block but we need css to understand them

divs and spans

  • <div> is the default block element

    has no special meaning at all.(used for division)
    It represents its children.
    It can be used with the class and ids (css styling)

  • <span> is the default inline element

    The span element doesn't mean anything on its own,
    but can be useful when used together with the global attributes class and ids
    It represents its children.
