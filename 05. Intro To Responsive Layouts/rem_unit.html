REM unit

is a multiplier for the root/html element size (font-size : 16px,100%;).

62.5%

16px represent 100% of the font-size of the document, if u want to keep everything simple computed, there is a practice using

  • font-size: 10px;
  • font-size: 62.5%

    • using this practice will make every rem size easier to calculate.

    rem & font-size chrome settings

    rem takes the absolute html/root element size as a reference, so changing the font-size chrome setting will make every other element size changes.

    Rem & font/padding/margin confusion.
