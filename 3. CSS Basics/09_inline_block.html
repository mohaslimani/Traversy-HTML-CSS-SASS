Inline & block

    we described before what is an inline and block

  • an inline is an element without any margin, border, padding and we cant fixat its width (because the width is a part of the padding).
  • a block have all those properties.
  • inlines are displayed one by one at the same line until we run-out the width of the window. (arranged from left to right)
  • blocks take by default the width of the window as their width and start always from the beginning of the line (arranged from top to bottom)
  • the most important shit here, u can change an element from being a block to inline and vise-versa (display)
