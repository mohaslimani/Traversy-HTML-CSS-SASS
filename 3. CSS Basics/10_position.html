CSS Position

Position : Static

    this is the default value for positioning and it means that the element is not affected by the properties like :
  • top : 100px
  • right : 100px
  • bottom : 100px
  • left : 100px
    • the red box is the static box

Position : relative

  • now u can use the (trbl) properties.
  • they depend to the parent element (static).
  • or we can say they depend to the supposed static place of the element itself.
  • top: 50px; / left: 50px;

    static-box

z-index : number;

  • z-index is a property that controls the accumulation between elements
  • by default the element will be in top of its previous
  • the z-index takes a number serve the importance
  • the higher the closer will be to u. (zero is the default)
  • the number can be positive or negative

Position : Absolute

  • u can use the (trbl) properties.
  • they depend to the parent element.
  • the parent element should be relative. else it will depend to the (first window) element.
  • for more fun try to zoom(in/out) or change the window height and width.
  • also (absolute position) is relative to the scroll.
  • top: 50px; / left: 50px;
  • bottom: 50px; / right: 50px;

Position : fixed

  • exist out of the flow of the body.
  • depend to the window.
  • fixed for all the other elements.
  • exist in top of other elements.
  • doesnt matter where exists in the doc. wouldnt effect the height of his parent.
    • fixed-box

Position : sticky

  • used for menus.
  • a fixed position element. to its original place in the doc.
  • depends on its parent element.
  • BUT
  • it sticks to the position u gave it when scrolling.
    • sticky-box
